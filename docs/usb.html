USB API

Speak HTTP-like commands over USB using a serial adapter.

Serial framing Pipe escaping

Framing

USB requests are sent as pipe-delimited frames. Escape literal pipes with \|.

METHOD|PATH|BODY

Example exchange

Send a request to list available formats:

POST|/api/formats/available|
GET|/api/version|

Responses are returned as plain text or JSON depending on the endpoint.
