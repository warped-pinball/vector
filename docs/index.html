Handler: check_password

Authentication: Required – see authentication guide.

Validate password proof

Request

Headers parameters

  • x-auth-challenge (hex-string, required) - Nonce provided by /api/auth/challenge
  • x-auth-hmac (hex-string, required) - HMAC signature over the challenge

Response

Status Codes

  • 200 - Credentials accepted
