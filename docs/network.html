Network access

Reach Vector over WiFi or Ethernet. All HTTP endpoints are POST-only unless noted otherwise.

POST-first TLS friendly

Base URL

Use your board's IP or hostname. Examples:

  • http://192.168.1.42
  • http://vector.local

curl examples

Fetch the current version (unauthenticated):

curl http://vector.local/api/version

Trigger a factory reset (authenticated):

curl -X POST \
  -H "x-auth-challenge: <token>" \
  -H "x-auth-hmac: <signature>" \
  http://vector.local/api/settings/factory_reset

TLS

Self-signed certificates are supported. Provide -k to clients like curl if you want to skip certificate validation during development.
